# Homework

1. Create a new branch: `git checkout -b context-rules`

2. Copy & paste the `instructions` and `prompts` directories into the `.github` directory
    - Don't overwrite any instructions or prompts that already exist!

3. Run: `/init-context`

4. Improve context in key areas
    ```text
    Gather more context about <AREA>. Summarize it for me and then update our @<FILE>.instructions.md file.
    ```

5. Cleanup the file for human readability

6. Customize a slash command

7. Push this to the repo
